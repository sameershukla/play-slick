This repo shows how Play and Slick integration works. It's 1-1 relationship between class Person and Address, Sample JSON required for input is 

{
    "personalInfo": {
        "name": "ABC",
        "email": "a@a.com",
        "age": 35,
        "sex": "Male"
    },
    "addressInfo": {
        "street": "A-1",
        "city": "ABC",
        "state": "ABC."
    }
}
